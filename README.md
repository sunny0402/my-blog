## About

- Blog website with React, Express and and MySQL.
- Heroku for deployment. ClearDB.
- Sequelize
- React Router v6.
- material-ui
- Markdown blog:
  - Blog articles are edited and saved in markdown format.
  - Saved to databse as a markdown string.
